HTML ÖĞREN - HTML Öznitelikleri

HTML Öznitelikleri

HTML öznitelikleri, elemanlar hakkında ek bilgi sağlar.

  • Tüm HTML elemanlarının öznitelikleri vardır.
  • Öznitelikler başlangıç etiketi içerisinde belirtilir.
  • Öznitelikler genelde isim/değer şeklinde belirtilirler. name=value

Lang Özniteliği

<html> etiketi içerisinde kullanılır.

Belgenin dil özelliğini belirtilir.

 <html lang="tr"></html>

Title Özniteliği

Elemanın üzerine fare ile geldiğinizde title özelliğinde belirttiğiniz metin ip ucu olarak görüntülenir.

<p title="bu bir paragraftır">Fare ile üzerine geliniz.</p>

Fare ile üzerine geliniz.

Bunu Aklından Çıkartma: Öznitelik tanımlamalarında tırnak işareti kullanmayı unutursan belge içerisinde hatalara sebep olabilir.

En Çok Kullanılan Öznitelikler

Öznitelik Açıklama
alt Görüntü için alternatif metin belirtir.
disabled Bir elemanın devre dışı kalmasını sağlar.
href Elamana tıklamadan sonra gideceği sayfayı/siteyi belirler.
id Eleman için benzersiz bir kimlik belirlemeyi sağlar.
src Görüntü için görselin arama yolunu belirtir
style Eleman için satır içi css yazmayı sağlar.
title Elemanın üzerine fare ile geldiğinizde title özelliğinde belirttiğiniz metin ip ucu olarak görüntülenir.

Bunu Aklından Çıkartma: Tüm öznitelikleri görmek için w3schools dan faydalanabilirsin.
