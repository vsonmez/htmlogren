HTML ID'leri

HTML id'leri bir html dokümanında benzersiz olmalıdır. Aynı id'ye sahip başka bir HTML elemanı olamaz. ID, HTML elemanın kimlik numarasıdır.

ID'ler css ve javascript içerisinde HTML elemanlarına çeşitli işlemler yapmak için kullanılır. tarayıcıda görüntülenmez.

Örneğin "link_kaydet" linkine tıkladığımızda bazı işlemler yapması için javascript içerisinde bu id'yi kullanırız.
