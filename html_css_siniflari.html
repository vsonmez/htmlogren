HTML CSS Sınıfları (Class)

HTML class'lari bir yada daha fazla HTML elemanında kullanılabilir.

Class'lar ile CSS dosyalarında HTML elemanı için çeşitli görsel tanımlamalar yapılır.

Örneğin "kirmizi_link" CSS sınıfına ait belirlenen görsel kurallar yukarıdaki link için tarayıcı tarafından uygulanır.
