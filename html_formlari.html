HTML Form'ları

HTML <form></form> elemanı, kullanıcıdan bilgiler almak amacıyla kullanılır. Örneğin kayıt işlemi yaparken yada kullanıcı girişi yaparken ihtiyaç duyarız.

<form></form> elemanı içerisinde, verileri almak için başka HTML Form Elemanları kullanılır; textbox, checkbox, radio button yada button gibi...

<form></form> alanlarının 3 ayrı özniteliği vardır:

"action" submit butonuna tıklandığında olması gereken işlem burada belirtilir. Genellikle, form içerisinde bulunan veriler JavaScript ile alınır ve işlem yapılacak başka bir sayfaya yönlendirilir.

"target" action bölümünde yapılan işlemin sonucunun nerede göseterilceği belirtilir. HTML Bağlantıları bölümünde anlattığımız "target" özniteliği ile aynıdır.

"method" action bölümünde yapılacak işlemin tipini belirtilir, "GET" ya da "POST" değerleri alır.
