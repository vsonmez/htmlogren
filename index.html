HTML ÖĞREN - HTML'in tanımı

HTML Nedir?

HTML, Web sayfaları oluşturmak için kullanılan standart bir döküman biçimlendirme dilidir. HTML dökümanlar ".html" ya da ".htm" dosya uzantısına sahiptir.

Örnek: "benimwebsayfam.html" ya da "benimwebsayfam.htm". Daha çok ".html" tercih edilir.

HTML'nin açılımı: Hyper Text Markup Language olarak geçmektedir. HTML, web sayfalarının yapısını oluşturmak için kullanılır ve HTML öğeleri sayfaların yapı taşlarıdır.

HTML öğeleri etiketlerle gösterilir, bu etiketler "p", "table", "h1", "h2", "div", "section" gibi isimler alırlar.

Tarayıcılar HTML etiketlerini göstermez. Tarayıcılar bu etiketleri yorumlar ve sayfanın nasıl görüneceğini anlamak için kullanır.

Örnek Bir HTML Dokümanı

<!DOCTYPE html> <html> <head> <title>Page Title'</title> </head> <body> tarayıcıda görüntülenecek içerik </body> </html>

Örneği İnceleyelim:

  • <!DOCTYPE html> Dökümanın HTML5 sürüm standartlarında olduğunu belirtir. ( HTML sürümleri ile ilgili detaylı açıklamalar alt bölümlerde mevcut. )
  • <html> Bir HTML dökümanı olduğunu belirtir.
  • <head> Döküman ile ilgili meta bilgilerini barındırır. (ileride detaylı inceleyeceğiz)
  • <title> Dökümanın başlığını belirtir. ( alt bölümlerde. görsel ile anlatılmıştır )
  • <body> Tarayıcı tarafından yorumlanıp görüntülenecek HTML etiketlerini içinde barındırır.( alt bölümlerde görsel ile anlatılmıştır. )

HTML Etiketleri

HTML etiketleri köşeli ayraçlarla(< >) çevrili öğe adlarıdır:<etiket> içerik... </etiket>

HTML etiketleri genellikle <p> ve </p> gibi çiftler halinde belirtilir. Bu çiftlerde ilk etiket başlangıç ​​etiketidir, ikinci etiket bitiş etiketidir. Bitiş etiketi başlangıç ​​etiketi gibi yazılır, ancak etiket adının önüne eğik çizgi / eklenir.

Bunu Aklından Çıkartma: Başlama etiketi açılış etiketi , sonlandırma etiketi kapanış etiketi olarak adlandırılır.

İnternet Tarayıcıları

Bir web tarayıcısının (Chrome, Internet Explorer, Firefox, Safari) amacı, HTML dökümanlarını okumak ve görüntülemektir. Tarayıcı, HTML etiketlerini görüntülemez, ancak bunları belgenin nasıl görüntüleneceğini belirlemek için kullanır.

DOCTYPE Tanımı

<!DOCTYPE> bildirimi belge türünü ve tarayıcıların web sayfalarını doğru şekilde görüntülemelerine yardımcı olur. Sayfanın üstünde (herhangi bir HTML etiketinden önce) yalnızca bir kez belirtilir.

<!DOCTYPE> bildirimi büyük / küçük harfe duyarlı değildir.

HTML için <!DOCTYPE> bildirimi şöyledir:

<!DOCTYPE html>
