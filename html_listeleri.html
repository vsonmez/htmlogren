HTML Listeleri

HTML listeleri maddeler halinde listeleme yapmamızı sağlar.

Sırasız bir liste, <ul> etiketi içerisinde<li> etiketiyle belirtilir.

<ul> <li> Liste Öğesi </li> <li> Liste Öğesi </li> <li> Liste Öğesi </li> <li> Liste Öğesi </li> </ul>

<ol> <li> Liste Öğesi </li> <li> Liste Öğesi </li> <li> Liste Öğesi </li> <li> Liste Öğesi </li> </ol>
